Informe sobre el Análisis de Datos de Arrestos en Ecuador

El informe actual analiza los datos de arrestos en Ecuador a partir de un archivo CSV. El mismo tiene información importante que nos permite comprender las diferentes tendencias relacionadas con las detenciones en nuestro país.

Como parte de preparación de los datos hemos realizado una limpieza para asegurarnos de que los datos estén listos para su análisis y procesamiento. Primero, hicimos la carga de la base de datos en nuestro entorno de Ubuntu, el cual es una plataforma muy útil para manejar grandes volúmenes de datos. Una vez cargados los datos, creamos una nota de Zeppelin, la cual facilita la realización de análisis de datos interactivos y visualizaciones.

En esta nota aplicamos todos los conocimientos adquiridos a lo largo de la materia. El primer paso fue configurar correctamente el intérprete para poder establecer una conexión segura con nuestra base de datos. Una vez configurado, hicimos varias consultas con MySQL. Estas nos permitieron extraer información importante de nuestra base de datos, la cual metimos en un DataFrame y con estos comenzamos a crear gráficos que nos permiten visualizar todo lo relacionado con las detenciones.

Utilizamos diversas librerías de Python, como Matplotlib y Seaborn, para realizar nuestras visualizaciones. Estos gráficos son esenciales para dar a conocer nuestros resultados y permitir a las demás personas tener una perspectiva de lo que sucede en nuestro país.

Desarrollo

Dado los datos que extraímos de la base de datos a partir de un CSV, con el objetivo de identificar las tendencias relacionadas con las detenciones del país.

Se ha realizado una serie de pasos para preparar, analizar y visualizar los datos de manera efectiva. Inicialmente, realizamos la limpieza de los datos; después, cargamos el CSV a nuestra base de datos en Ubuntu. Al ser una plataforma muy buena para manejar grandes cantidades de datos, creamos notas que nos permitieron manejar los datos.

Repositorio de GitHub: https://github.com/JeanDavidVasquez/ProyectoIntegrador.git

Herramientas utilizadas en la máquina Virtual:

  • Actualizar el sistema:
    • sudo apt-get update
    • sudo apt-get upgrade
  • Instalar Python 3:
    • sudo apt-get install python3
  • Instalar MySQL:
    • sudo apt-get install mysql-server
  • Instalar pip:
    • sudo apt-get install python3-pip
  • Instalar las librerías de Python necesarias:
    • pip install pandas
    • pip install matplotlib
    • pip install seaborn
    • pip install sqlalchemy
  • Instalar Zeppelin:
    • Descargar la versión más reciente desde el sitio web de Apache Zeppelin.
    • Descomprimir el archivo descargado y seguir las instrucciones de instalación.
